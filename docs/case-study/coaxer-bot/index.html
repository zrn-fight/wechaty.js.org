WeChat Daily Talk is a WeChat little love talk tool based on node and wechaty. Initially, the function was only to send the weather and a sentence of love every day. Later, the intelligent robot chat function was added, automatically adding groups, automatically adding friends, and timing assistant functions.

However, because this project is for Xiaobai users and users who are new to node development, two projects have been split, one is the function-specific "WeChat Daily Talk" (that is, this project) for Xiaobai, and the other is also "Smart Secretary" under Leo's warehouse is for users with more programming experience.
