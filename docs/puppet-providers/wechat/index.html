Puppet Provider: WeChat

  • Repo: https://github.com/wechaty/wechaty-puppet-wechat
  • Support & Feedback: https://github.com/wechaty/wechaty-puppet-wechat/issues
  • Go Wecahty and WeChat Web Protocol, @dchaofei, Apr 16, 2021

Features#

This is a full-featured Wechaty Puppet.

Known Issues#

WeChat Accounts that are registered after 2017 might not be able to login Web Wechat, so it can not use wechaty-puppet-wechat with Wechaty. Please make sure your WeChat Account is eligible to login by visiting Web WeChat

Few points to be kept in mind :

  1. Web API can not create room and invite members to room since 2018.
  2. The ID of contacts and rooms will change across sessions. #1644

If you want to break the above limitations, then you need consider to use a Wechaty Puppet other than Web. Learn them from Wechaty Puppet Providers.

Fixed Issues#

  1. Support UOS with puppeteer : #127

  • Linux
  • macOS
  • Windows
